一.登陆系统
   1.用户分普通用户和管理员用户。
   2.普通用户:hanxin 密码:123456。
   3.管理员用户:administrator 密码:123456
   4.管理员账户除了可以使用用户管理功能外，与普通用户无区别。
二.上传文件。 put filename 
   1.客户端输入:put test.txt
   2.判断本地文件是否存在。
     ①.存在，执行下一步
     ②.不存在，提示文件不存在
   3.判断服务器中文件是否已经存在.(通过MD5值判断)
     ①.不存在，执行下一步上传文件。
     ②.存在，返回服务器上存在的文件名称。
   4.判断是否可以续传.
     ①.输入1,重新上传.
     ②.输入2,续传.
   5.不可续传，直接上传文件。
   6.接收服务器返回的上传状态。
   7.客户端打印相关信息。
三.下载文件。 get filename
   1.客户端输入:get test.txt
   2.判断服务器中文件是否存在.(通过文件名值判断)
     ①.不存在，返回不存在信息。
     ②.存在，向客户端发送文件。
   3.客户端打印相关信息。
四.列出所有文件 list
   1.客户端输入:list
   2.服务器查询数据库，并返回给客户端结果。
   3.客户端打印相关信息。
五.删除文件 delete filename
   1.客户端输入:delete test.txt
   2.判断服务器中文件是否存在.(通过文件名值判断)
     ①.不存在，返回不存在信息。
     ②.存在，删除文件，并从数据库中删除文件信息。
     ③.反馈给客户端信息。
   3.客户端打印相关信息。
六.重命名文件:rename old_filename new_filename
   1.客户端输入：rename test.txt new_test.txt
   2.判断服务器中文件是否存在,并判断新文件名是否被占用.(通过文件名值判断)
     ①.如果输入的旧文件名存在，并且新文件名未被占用，则重命名，并修改数据库信息。
     ②.否则，打印错误信息。
   3.客户端打印相关信息。
七.修改密码:repasswd newpasswd newpasswd_again
   1.客户端输入：repasswd testpassword testpassword
   2.服务器端修改数据库中该用户对应的密码，并向客户端返回结果
   3.客户端打印相关信息。
八.用户管理(仅限管理员使用):user
   1.客户端输入:user
     ①.用户为普通用户，则提示无权限使用该命令。
     ②.用户为管理员用户
       (1)增加用户
           .1客户端输入:useradd  testuser testpassword testpassword  
                     （useradd命令后为用户名和密码，长度均需大于等于6位)
           .2如果数据库中目前没有testuser用户名，则服务器端在数据库中添加用户及对应的密码，并向  客户端返回结果
           .3客户端打印相关信息。
       (2)删除用户
           .1客户端输入:  deluser testuser
           .2如果数据库中目前有testuser用户名,从数据库中删除该用户，并返回信息
           .3客户端打印相关信息。
       (3)列出所有账户
           .1客户端输入：userlist
           .2服务器端返回user信息
           .3客户端打印相关信息。
       (4)返回上一层
          客户端输入：back，则返回上一层功能表
       (5)退出系统
          客户端输入：quit，则退出整个系统。
九.退出系统:quit
   1.客户端输入:quit
   2.退出系统
